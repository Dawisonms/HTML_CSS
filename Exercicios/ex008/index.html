Negrito e Itálico

Importancia de usar o Negrito usando o <strong> que é semântico ao invés de usar <b>, que não é semântico

Da mesma forma, é importante usar o <em> para itálico, que é semêntico, ao invés de usar o <i>, não semântico

Usando i com atalho Ctrl + Shift + p (Procurando Wrap with abbreviation)
Usando em com atalho Ctrl + Shift + p (Procurando Wrap whit abbreviation)
Importante selecionar a frase antes de usar o comando acima.

Marcação de texto

Podemos usar uma marcação de texto usando o mark. Caso queira alterar a cor (padrão em amarelo, você podera criar um slyle em head como nesse projeto)

Textos Grandes e Pequenos

Podemos fazer textos grandes usando <big>, porém está obsoleto e devemos mudar a forma...Pesquisar qual...

Para textos pequenos, podemos usar o <small>.

Textos excluidos

Para textos excluidos, podemos usar o <del>

Textos iseridos ou sublinhados

Podemos usar textos inseridos ou sublinados usando o <ins>. Não devemos usar o <u> para sublinhar, pois está obsoleto.
